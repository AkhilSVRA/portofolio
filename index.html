JAVA-Based Rock-Paper-Scissors Game

I developed a Rock Paper Scissors game in Java, where users can play against the computer in a console-based environment. The game features simple user input handling, random number generation for the computer's moves, and logic to determine the winner based on traditional rules. It includes input validation to ensure smooth gameplay and allows users to replay multiple rounds in the same session. This project helped me strengthen my understanding of Java, particularly in the areas of conditional statements, loops, and user interaction.

Medical Robot

In my B.Tech, I developed a major project focused on creating a medical robot designed to assist healthcare environments. The robot was equipped with the capability to serve tablets to patients and monitor their activities through a network of sensors, significantly reducing the need for direct social interactions between patients and healthcare staff. Utilizing Wi-Fi for real-time data transmission, the system aimed to improve patient care efficiency while maintaining safety protocols, especially in environments requiring minimal physical contact.
